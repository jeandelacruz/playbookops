# Guía De Revisión

Los revisores deben comprobar:

- El caso de uso es claro.
- Los permisos son mínimos.
- La política de evidencia evita conclusiones inventadas.
- El output contract es completo y verificable.
- La salida esperada describe estilo, secciones y chequeos de calidad.
- Las safety rules bloquean acciones destructivas sin confirmación.
- Las recomendaciones incluyen validación y limitaciones.
- El scorecard es razonable.
