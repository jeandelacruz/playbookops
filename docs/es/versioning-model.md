# Modelo De Versionado

PlaybookOps usa versionado semántico para el catálogo y para cada playbook.

- Patch: correcciones de texto, metadata o documentación.
- Minor: nuevas secciones, adapters o validaciones compatibles.
- Major: cambios de intención, cambios de ejecución o cambios incompatibles de schema.

Todo cambio de playbook debe añadir una entrada al changelog.
