# Modelo De Madurez PromptOps

- `L0`: prompt crudo.
- `L1`: prompt estructurado.
- `L2`: prompt con output contract.
- `L3`: playbook portable para varias herramientas.
- `L4`: playbook evaluado con ejemplos.
- `L5`: playbook revisado y listo para comunidad.

La meta de PlaybookOps es llevar prompts utiles hacia `L3` como minimo y preparar los mejores para `L4` o `L5`.
